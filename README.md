# Menu com While e For em Python

Este é um programa simples em Python que utiliza estruturas de repetição `while` e `for`.  
O usuário interage com um menu que permite:

1. Contar de 1 até um número escolhido  
2. Repetir uma palavra várias vezes  
3. Sair do programa  

O programa só termina quando o usuário seleciona a opção de sair.
